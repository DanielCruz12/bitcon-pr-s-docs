Books and Resources for Bitcoin Core

Here are a few books and resources I have found helpful for understanding Bitcoin Core.

C++

Bitcoin Core is currently written in C++11 and will be transitioning to C++17 starting from November 2020. Here are some useful resources:

The Reference

  • The C++ Programming Language, 4th Edition

Online resources

  • C++ reference
  • C++ Core Guidelines by Bjarne Stroustrup and Herb Sutter
  • Bitcoin Core PR Review Club
  • Bitcoin Core Developer Notes
  • Godbolt Compiler Explorer
  • Google C++ Style Guide
  • C++ FAQ

Learning C++

  • C++ Primer, 5th Edition (more gentle, longer, not to be confused with the knock-off, "C++ Primer Plus")
  • Accelerated C++ (condensed, faster, a bit out of date but excellent, exercise solutions here)

Intermediate/advanced

  • Effective C++
  • Effective Modern C++
  • Effective STL
  • Exceptional C++
  • More Exceptional C++
  • C++ Templates: The Complete Guide (1st Edition)
  • C++ Templates: The Complete Guide (2nd Edition)
  • C++ Concurrency in Action, 2nd Edition

Cryptography

  • Elliptic Curve Cryptography: a gentle introduction, a blog series by Andrea Corbellini
  • Foundations of Cryptography
  • Waxwing's blog by the cryptographer and creator of JoinMarket, Adam Gibson
  • A Graduate Course in Applied Cryptography by Dan Boneh
  • Cryptopals - A set of cryptographic code challenges
